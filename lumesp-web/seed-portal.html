Why this matters: these inboxes build trust with Gmail, Outlook & Yahoo so Lume's emails land in real inboxes instead of spam. Set one up once and our system runs it forever — you never have to open it again.
Do this for each account — just 3 steps
1
Create the email account. Make a new Gmail, Outlook, or Yahoo account. Use a real-looking name (like firstname.lastname) and add a recovery phone when asked — that keeps it from getting locked.
Tip: don't make them all at once on the same Wi-Fi. Spread them out — some at home, some on your phone's data — over a day or two. 2–3 at a time, then a break.
2
Turn on 2-step verification, then create an "app password." This is a one-time code that lets our system connect safely — it is not your normal password.
Gmail: Account → Security → turn on 2-Step Verification → search "App passwords" → generate one. Also turn on IMAP (Settings → "Forwarding and POP/IMAP").
Outlook: account.microsoft.com → Security → Advanced security → turn on two-step → "Create a new app password."
Yahoo: Account Security → turn on two-step → "Generate app password" → choose "Other app."
3
Paste it below. Pick the provider, type the email address and the app password. A green ✓ means it connected — and the bars above climb toward 99. That's it!
