# `playbooks/` — reference playbooks

The "how we do it" references — not setup steps, not per-campaign runbooks. These are stable
documents you consult while building features or writing copy.

| File | What it covers |
|---|---|
| **daily-bd-operating-model.md** | The pared-down BD funnel: what runs automatically, the one-time activation checklist, and the 5-step / 20-min daily loop. |
| **copywriting-playbook.md** | The persuasion playbook (Bernays-style) applied to site copy and outreach messaging. The voice/guidelines the message drafters follow. |
| **website-map.md** | The full map of every page on the site + where things live — a content/IA reference for the marketing site and portal. |
