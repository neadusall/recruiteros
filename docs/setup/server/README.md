# `setup/server/` — get the app live

Everything to take RecruitersOS from nothing to a working HTTPS URL. Follow the numbered guides in
order for a fresh install.

| File | What it covers |
|---|---|
| **1-hetzner.md** | Initial server setup — Ubuntu, Docker, Caddy, firewall on a Hetzner box. |
| **2-deploy.md** | Getting a live URL — local dev, Vercel, env vars, and persistence notes. |
| **3-go-live-walkthrough.md** | End-to-end go-live — DNS records, Caddy auto-HTTPS, the deploy script. |
| **integration-architecture.md** | How outreach works under the hood — the three-layer architecture and the three execution modes. Read this to *understand* the system (not a setup step). |
