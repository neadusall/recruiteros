# `setup/` — stand up and configure RecruiterOS

Step-by-step guides for getting the platform **running** and **connected to its channels**. If you're
deploying for the first time or switching on a new outreach channel, you're in the right place.

## Subfolders

| Folder | What it's for |
|---|---|
| **[server/](server/)** | Get the app **live**: provision the server, deploy, go live, and understand the integration architecture. Numbered in the order you'd follow them. |
| **[channels/](channels/)** | Turn on each **outreach channel**: cold email, transactional email (Resend), SMS, and LinkedIn login. One guide per channel — do only the ones you need. |

> Rough order for a brand-new install: `server/1-hetzner` → `server/2-deploy` →
> `server/3-go-live-walkthrough`, then add channels from `channels/` as you need them.
