# `setup/channels/` — turn on outreach channels

One guide per channel. Each is independent — set up only the channels you plan to use. You can add
more later without redoing anything.

| File | What it covers |
|---|---|
| **cold-email.md** | Cold outbound email campaigns — full walkthrough of the sending setup. |
| **email-resend.md** | Transactional email via Resend — domain verification + API key (powers password reset & magic links). |
| **sms-qualification.md** | Two-way SMS via Telnyx + the intent classifier that qualifies replies. |
| **linkedin-login.md** | LinkedIn OAuth app + callback configuration (Sign in with LinkedIn). |
