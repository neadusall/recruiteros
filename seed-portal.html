Why we're doing this: these inboxes quietly build sending reputation so our real emails land in the inbox, not spam.
  • Create each account, turn on 2-step verification, and generate an app password (a one-time code that lets our server connect, not your normal password).

  • Paste each address + app password below. We test the connection instantly and show a green check.
How to generate an app password (Gmail · Outlook · Yahoo)
  1. Gmail: Account → Security → turn on 2-Step Verification → search "App passwords" → generate one → also enable IMAP in Gmail Settings → "Forwarding and POP/IMAP".
  2. Outlook / Hotmail: account.microsoft.com → Security → Advanced security options → turn on two-step verification → "Create a new app password".
  3. Yahoo: Account Security → turn on two-step verification → "Generate app password" → pick "Other app".
Paste the app password exactly as shown (spaces are fine, we strip them).
